  ---
  ❌ POINTS NÉGATIFS / BLOQUANTS

  🔴 CRITIQUE — Sécurité

  La clé API Gemini est exposée dans le bundle public via vite.config.ts :
  define: { 'process.env.GEMINI_API_KEY': JSON.stringify(env.GEMINI_API_KEY) }
  N'importe qui peut ouvrir les DevTools et l'extraire. À corriger avant tout déploiement.

  ---
  🔴 CRITIQUE — App Store : l'app n'est pas soumettable telle quelle

  C'est une PWA web, pas une app native. Pour l'App Store Apple ou le Play Store Google, il faut obligatoirement :
  1. Wrapper l'app avec Capacitor (recommandé) ou Cordova pour la transformer en app native
  2. Créer des icônes aux formats requis (1024×1024 pour iOS, multiples tailles pour Android)
  3. Signer le build avec un certificat Apple Developer ($99/an) ou Google Play ($25 une fois)
  4. Passer la review Apple (7-14 jours) / Google (quelques heures)

  Sans Capacitor, la seule option est l'installation PWA depuis le navigateur (ajout à l'écran d'accueil) — mais ce n'est pas l'App Store.

  ---
  🔴 CRITIQUE — Pas de manifest PWA ni de Service Worker

  Sans manifest.json :
  - L'app ne peut pas être installée depuis Chrome/Safari
  - Pas d'icône sur l'écran d'accueil
  - Pas de mode plein écran sans la barre du navigateur

  Sans Service Worker :
  - Zéro fonctionnement hors-ligne — une coupure réseau = écran blanc
  - Les notifications push en arrière-plan sont impossibles

  ---
  🟠 MAJEUR — Dépendances parasites

  express est dans les dependencies (production) alors que c'est un package serveur. Il gonfle le bundle inutilement.

  framer-motion et motion sont tous les deux présents — motion est le nouveau nom de framer-motion v12. Doublons à nettoyer.

  Le nom du package est encore "react-example" dans package.json — résidu du projet starter.

  ---
  🟠 MAJEUR — Pas de tests

  Zéro test unitaire ou d'intégration. Sur une app Firebase avec logique métier (XP, cartes, challenges), un bug peut passer en prod sans être détecté.

  ---
  🟡 MINEUR — Fonctionnel incomplet

  - category existe dans les types (famille / ami / autre) mais n'est jamais utilisé dans l'UI
  - Seulement 2 challenges — le système est sous-exploité
  - Pas de recherche / filtre dans la liste d'amis — problématique dès 30+ amis
  - Célébrités incomplètes sur 11 mois sur 12
